Consignes de mon exercice réaliser avec WampServer :

Écrire trois tableaux associatifs avec des valeurs pour les clés : "nom", "prénom" et "naissance".
Définir un tableau de ces tableaux.
Définir une fonction age() qui calcule la différence en années entre deux dates.
Si la deuxième date est absente, prendre la date d'aujourd'hui.
Utiliser cette fonction pour afficher l'âge de chaque personne dans le tableau.
Afficher dans un tableau HTML les données de ces tableaux.
Les clés du tableau représentent les en-têtes du tableau.
Le tout dans une page HTML valide, bien évidemment.
