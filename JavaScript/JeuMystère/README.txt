Création d'un programme qui récupère un mot tapé dans un formulaire HTML et le transforme en code secret.
La première étape consiste à définir aléatoirement le code à utiliser. 
Chaque lettre de l'alphabet est associée à un nombre entre 1 et 26. 
La deuxième étape consiste à récupérer le mot saisi via un formulaire. 
Enfin, la dernière étape consiste à générer le code secret en fonction du mot saisi dans le formulaire et des lettres associées à chaque nombre, puis à afficher le résultat dans la page HTML.
